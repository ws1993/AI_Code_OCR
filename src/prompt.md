## Rules

1. **易混淆与重复字符处理**：

   - `l/1`：变量名中优先 `l`，数字中优先 `1`
   - `0/O`：十六进制中优先 `0`，类名中优先 `O`
   - `;/:`：行尾优先 `;`，条件语句后优先 `:`
   - `./,`：浮点数优先 `.`，参数分隔优先 `,`
   - **重复符号还原**：如连续 `;;`、`::`、`==`、`--` 等，需严格还原原始数量，禁止合并或遗漏
   - *使用语法树和字符坐标动态校验*

2. **换行处理**：

   - **保留换行**：当行尾有 `\`、`,`、`{`、运算符或连续符号时
   - **取消换行**：当满足以下条件时合并行：
     * 下一行缩进量 <= 当前行基准缩进
     * 当前行未结束（缺少 `;`或 `}`）
     * 合并后长度 < 120 字符

3. **格式保留**：

   - 严格保持原始缩进（制表符/空格不转换）
   - 注释位置 100%还原
   - 保留连续空行（≥2 行）

